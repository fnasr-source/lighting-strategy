# Simulations Labs — Report Information Architecture

_Generated: 2026-03-01_

## Narrative Flow
Executive-first → Evidence-second → Plan-third

## Section Hierarchy

### 1. Executive Summary (one-screen)
- **What's done**: Key deliverables completed
- **Key findings**: Top 3–5 data-backed insights
- **Main blockers**: Analytics gaps, tracking disabled
- **Top next actions**: Prioritized immediate steps
- Confidence badge: section-level

### 2. Work Completed Snapshot
- Workstream cards: Outreach, Operations, SEO/Content, Meetings, Lead Pipeline
- Each card: status (done/in-progress/pending), key metric, milestone dates
- Timeline visualization: May 2025 → Mar 2026

### 3. Evidence Dashboard
- **Outreach Performance**: sent/replied/bounced per campaign (bar chart)
- **Lead Pipeline**: status distribution (donut), priority breakdown (bar), source analysis (bar)
- **Operations**: task completion by project (progress bars), link health (pie)
- **Meeting Cadence**: meeting frequency over time (line/area chart)
- Every chart: source file reference + confidence badge

### 4. Insights & Data Gaps
- Split layout: validated insights (left) + open questions (right)
- Data gaps table with impact and resolution path
- Risk matrix: impact × likelihood for unresolved gaps

### 5. Forward Plan
- 30/60/90 day plan with milestones
- Dependencies + expected outcomes
- Decision points requiring stakeholder input
- Source coverage outlook chart

### 6. Appendix
- Source traceability table (full 01_SOURCE_MAP)
- Data dictionary (from 02_DATA_DICTIONARY.md)
- Methodology notes
- Glossary (Instantly, Asana, Fathom, Systeme terms)

## Visual Components Required
1. Workstream status chart (horizontal stacked bar)
2. Milestone timeline (horizontal timeline with icons)
3. Risk matrix (2×2 or 3×3 grid)
4. Source coverage chart (radar/bar)
5. Meeting frequency chart (area chart by month)
6. Confidence badges (High=green, Medium=amber, Low=red) per section
7. KPI cards (4–6 top-level metrics)
8. Lead pipeline funnel or sankey visualization
