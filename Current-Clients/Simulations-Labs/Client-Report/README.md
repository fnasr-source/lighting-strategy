# Client Report — README

This folder contains the Simulations Labs client-facing visual report, built as a static web application deployable via GitHub Pages.

## Quick Start
Open `index.html` in any modern browser. No build step required.

## Structure
- `index.html` — Main balanced report (recommended for sharing)
- `styles.css` — Brand-consistent stylesheet
- `app.js` — Chart rendering and interactivity
- `data/report-data.json` — All report data (derived from source files)
- `v1-executive-light/` — Minimal board-level summary
- `v2-balanced/` — Strategy + evidence (links to main report)
- `v3-deep-dive/` — Analyst-level with full appendix
- `docs/` — Methodology, assumptions, pipeline documentation
- `00_AUDIT_LOG.md` — Data audit results
- `01_SOURCE_MAP.md` — Chart-to-source traceability
- `02_INFORMATION_ARCHITECTURE.md` — Report structure plan
