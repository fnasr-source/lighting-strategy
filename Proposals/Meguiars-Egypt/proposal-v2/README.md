### Slide Structure (AIDA)

| Act | Slides | Key Message |
|-----|--------|-------------|
| **ATTENTION** | 1-4 | "Meguiar's doesn't exist in Egypt" — $75M opportunity, zero digital leaders |
| **INTEREST** | 5-9 | 3-Phase growth engine: E-Commerce → Home Service → B2B |
| **DESIRE** | 10-12 | Roadmap, 454K EGP value stack, modular pricing (32K/mo + add-ons) |
| **ACTION** | 13-16 | Risk reversal guarantee, credibility, getting started, closing |
