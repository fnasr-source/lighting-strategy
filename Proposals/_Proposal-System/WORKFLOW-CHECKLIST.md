# Proposal Workflow Checklist

1. Create or identify client folder in `Proposals/`.
2. Assign proposal number using `proposal-registry.csv`.
3. Add proposal number to:
   - `00-Proposal-Index.md`
   - `06-Offer-and-Proposal.md`
   - `10-One-Page-Proposal.html`
   - client email draft in `communications/`
4. Keep the package in numbered sequence (`00` to `11`).
5. Update one-page proposal and final presentation with matching options.
6. Confirm payment section by region:
   - Egypt: Instapay
   - International: payment link
7. Add final links to email draft.
8. Push and share both URLs (presentation + one-page proposal).
