# Source Pattern

This starter combines two existing Admireworks project patterns.

## Strategy / Documentation Pattern
Primary influence: The Slim Game
- stronger strategy systemization
- stronger knowledge-base structure
- stronger client-facing collaboration pattern
- stronger strategy presentation pattern

## Operations / Admin Pattern
Primary influence: Edrak Space
- stronger onboarding and handover docs
- stronger campaign operations notes
- stronger WhatsApp operationalization pattern

## Research Pattern
Primary influence: Internal AW SOP
- structured market research before strategy
- competitor scan discipline
- browser and source-backed synthesis before positioning decisions

## Rule
Do not copy either source project blindly. Use this starter as the curated baseline and only add modules when a client actually needs them.
