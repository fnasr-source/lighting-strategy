# Bilingual Channel Adaptation Example

## Core Message
- Promise: clearer path, less friction, stronger buyer confidence
- Proof type: process clarity plus visible proof assets
- CTA direction: low-friction conversation first

## Ad Example
### English
People do not ignore unclear offers because they are not interested.
They ignore them because they do not trust what happens next.

### Arabic
الناس لا تتجاهل العرض لأنهم غير مهتمين.
هم يتجاهلونه لأن الخطوة التالية ليست واضحة.

### Note
- Same strategic idea.
- Different sentence shape so both versions sound native.

## Landing Page Example
### English
Show them what happens first, what they get, what proof exists, and what risk is removed.

### Arabic
وضّح لهم أول خطوة، ماذا سيحصلون عليه، ما الدليل، وكيف أزلت عنصر المخاطرة.

### Note
- The Arabic keeps command energy without sounding translated.

## WhatsApp Example
### English
If useful, I can send the short breakdown first so you can see exactly how it works before we talk.

### Arabic
إذا مناسب لك، أرسل لك الملخص المختصر أولاً حتى ترى بالضبط كيف يتم الموضوع قبل أن نتكلم.

### Note
- WhatsApp stays conversational in both languages.
- CTA is friction-aware and context-light.
