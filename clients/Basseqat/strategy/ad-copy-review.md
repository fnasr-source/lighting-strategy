# Ad Copy Review Framework

## Purpose
This file defines the ad angle system, variants per angle, and review criteria for Basseqat's paid media campaigns.

---

## Review Criteria per Ad

| Criterion | Check |
|---|---|
| Source truth | Does it rely only on confirmed facts from client KB or research? |
| Audience fit | Is it written for Abu Ahmad (40+, Egyptian expat, KSA)? |
| Market-native language | Is the Arabic natural Egyptian, not MSA or translated? |
| Offer clarity | Can the prospect understand what Basseqat is within 3 seconds? |
| Proof support | Is any claim backed by evidence or softened appropriately? |
| Objection awareness | Does it anticipate why someone might scroll past? |
| CTA fit | Does the CTA match the awareness stage? |
| Compliance | No guaranteed returns, no hype, no pressure? |
| Human quality | Would a native-speaking strategist approve this? |
