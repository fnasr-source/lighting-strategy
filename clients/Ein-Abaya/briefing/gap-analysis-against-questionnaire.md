## Extra Operational Fields Added For This Project

The standard questionnaire does not fully cover the operational needs of a Zid website kickoff, so the public onboarding form also includes:

- Zid access details
- domain / DNS ownership
- payment gateway and BNPL setup
- shipping and logistics setup
- analytics / ads access readiness
- policy links or status
- brand assets folder link
- media folder link
- catalog export link
- daily approver details

These are intentionally added because they are necessary for a fast Zid build and launch.
